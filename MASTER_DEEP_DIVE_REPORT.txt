### 2. WHAT REMAINS (CRITICAL PATH TO LIVE DEMO)
1. **Configure `.env` file**: Copy `.env.example` to `.env` and enter real credentials for `GEMINI_API_KEY`, `MEM0_API_KEY`, `NEO4J_URI`, `NEO4J_USERNAME`, and `NEO4J_PASSWORD`. *Blocker: Startup verification gate in `core/startup.py` blocks application launch if critical keys are missing.*
2. **Ingest PDF documents**: Place 3–5 PDF files in the `data/documents/` directory. *Blocker: Without files, LlamaIndex PropertyGraphIndex RAG engine defaults to empty graph and falls back to pure LLM chat.*
3. **Resume Neo4j Database**: Access the Neo4j AuraDB dashboard and ensure the database instance is in the "Running" state. *Blocker: Paused AuraDB instance prevents RAG engine connections, forcing RAG queries to fail.*
4. **Execute Activator**: Run `powershell -ExecutionPolicy Bypass -File activate.ps1` in the project root. *Blocker: Manual terminal execution required.*
